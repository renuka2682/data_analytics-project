# data_analytics-project
INTRODUCTION:
Customer churn is one of the most critical challenges faced by subscription-based businesses such as telecom and service industries. This project focuses on analyzing customer churn data to identify the key factors influencing customer attrition.
The analysis was performed using SQL for structured querying, Python for exploratory data analysis (EDA), and Power BI for interactive dashboard visualization. The objective is to transform raw customer data into meaningful insights that help improve retention strategies.

OBJECTIVES:
Calculate overall churn rate
Analyze churn based on contract type
Study the impact of monthly charges on churn
Examine the relationship between customer complaints and churn
Build an interactive Power BI dashboard for business insights

TOOLS AND TECHNOLOGIES USED:
SQL (MySQL) – Data querying and aggregation
Python – Data cleaning and exploratory analysis
Pandas
Matplotlib
Seaborn
Power BI – Interactive dashboard creation
MS Excel – Dataset handling
